Current Projects
  • Immediate Feedback in Blocks Based Environments
      This project uses analysis of novice programmer data to craft feedback. Using principles of Instructional Design and the Knowledge Learning Instruction (KLI) Framework, I'm identifying student misconceptions at a fine enough granularity to suggest actual interventions to class curriculum. The work includes the following:
        An intervention in a Computational Thinking class in blocks based environments using BlockPy
        Created a replay mechanism to replay students' programming sessions
        Developed a systematic method to improve an existing unit of instruction using cognitively-based framework of learning (KLI) that is particularly suited for the design of computing instruction
